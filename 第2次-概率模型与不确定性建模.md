## 开场：概率不是算命

上节课我们用两条生产线扩产问题做了一次数学语言体检。

你已经看到：平均值不是决策本身，波动不是装饰信息，损失结构会改变行动选择。

今天我们进入第二个问题：

> 当我们还没有足够数据时，怎样用概率语言表达判断？

请先在纸上写下今天的课堂契约：

```text
概率不是算命。
概率是在信息不完整时，对不确定性的结构化表达。
```

再写下这一句：

```text
概率模型不是现实本身，而是对现实生成过程的简化描述。
```

今天每一段仍按同一个节奏推进：

```text
先写判断 -> 看结果空间 -> 改成频数 -> 加入模型 -> 说明边界
```

到下课时，你要能说清五件事：

1. 为什么“连续 5 件合格”不是一句完整证据。

3. 频率解释和可信程度表达怎样互相补充。
4. 独立性假设为什么强大，也为什么危险。
5. 为什么下一课要讨论“增加信息以后，总体是否已经改变”。

此刻你应该能说清：

```text
今天不是练习套概率公式，而是练习建立概率模型。
```

## 先做判断

某材料厂有一批产品。质检员从这批产品中连续抽出 5 件，结果 5 件全部合格。

现在管理者问：

> 这一批产品质量是否很好？能否减少后续抽检？

请先不要计算。请在纸上写：

```text
第一次判断：质量很好 / 不能判断 / 需要继续抽检
理由：
还需要的信息：
```

停 1 分钟。

现在回答三个问题：

1. “连续 5 件合格”让你更放心，还是仍然不放心？
2. 你默认这 5 件产品是怎样抽出来的？
3. 如果这批产品有 10000 件，5 件全合格对你意味着什么？

先把三个词放在纸上：

```text
结果：看到的事实
模型：解释结果如何产生
边界：这个结果不能说明的部分
```

如果你刚才选择“质量很好”，你默认 5 件样本已经能代表整批产品。

如果你刚才选择“不能判断”，你已经意识到样本量、抽样方式和缺陷分布都没有交代。

如果你刚才选择“需要继续抽检”，你已经把“看到好结果”和“采取少检行动”分开。

此刻你应该能说清：

```text
5 件全合格是一个结果，不是完整结论。
```

## 列出结果

先不谈公式，只列出 5 件产品会出现什么结果。

每一件产品只有两种记录：

```text
Q = 合格
N = 不合格
```

如果只抽 1 件，结果空间是：

```text
{Q, N}
```

如果连续抽 2 件，结果空间是：

| 编号 | 第1件 | 第2件 | 结果 |
| ---: | --- | --- | --- |
| 1 | Q | Q | QQ |
| 2 | Q | N | QN |
| 3 | N | Q | NQ |
| 4 | N | N | NN |

请在纸上写下：连续抽 3 件时，一共有多少种结果？

停 1 分钟。

标准答案：

```text
2 × 2 × 2 = 8 种
```

连续抽 5 件时：

```text
结果数 = 2^5 = 32
```

现在给这些对象命名：

```text
样本空间 Ω：所有会被模型考虑的结果集合
事件 A：我们关心的一组结果
```

在这个问题里：

```text
Ω = 所有由 Q 和 N 组成的 5 位序列
A = 5 件全合格 = {QQQQQ}
```

请把下面这句话补完整：

```text
“连续 5 件合格”不是样本空间，它是样本空间里的一个 ________。
```

标准答案：

```text
事件
```

现在做一个小练习。

事件 B 定义为：

```text
B = 5 件里至少有 1 件不合格
```

请回答：

```text
A 和 B 是否会同时发生：
A 和 B 合在一起是否覆盖全部结果：
```

标准答案：

```text
A 和 B 不会同时发生。
A 和 B 合在一起覆盖全部结果。
```

此刻你应该能说清：

```text
概率模型的第一步不是计算，而是说清哪些结果被放进模型。
```

## 换成频数

现在不要急着写 `P(A)`。

先把概率换成更直观的频数。

假设有很多批产品，每一批都连续抽 5 件。我们问：

> 如果真实单件合格率不同，100 批里大约会有多少批出现“5 件全合格”？

先看表：

| 单件合格率 | 5件全合格的概率 | 100批中全合格批数 |
| ---: | ---: | ---: |
| 80% | 32.8% | 约 33 批 |
| 90% | 59.0% | 约 59 批 |
| 95% | 77.4% | 约 77 批 |
| 98% | 90.4% | 约 90 批 |

请先不要看公式。只看这张表，回答：

```text
如果单件合格率只有 90%，100 批里有多少批也会出现 5 件全合格？
```

标准回答：

```text
约 59 批。
```

这句话很重要。

> 5 件全合格不只会出现在质量极高的批次里。  
> 在单件合格率 90% 的模型下，它也经常出现。

现在把表格背后的公式写出来：

```text
如果每件产品合格概率为 p，且每件结果相互独立：

P(5件全合格) = p × p × p × p × p = p^5
```

代入：

```text
p = 0.90
P(5件全合格) = 0.90^5 = 0.59049
```

这不是说真实世界一定严格等于 59.049%。

这句话的含义是：

```text
在“单件合格率为 90%，每件独立”的模型里，
连续 5 件合格并不稀奇。
```

请把第一次判断改写一次：

```text
第二次判断：质量很好 / 不能判断 / 需要继续抽检
我改变或保留判断的原因：
```

此刻你应该能说清：

```text
概率把一个孤立结果，放回一组可比较的生成模型里。
```

## 加入模型

现在我们回到公式。

刚才使用的核心假设是：

```text
每件产品合格概率相同。
每件产品结果相互独立。
```

这两个假设合起来，才能把 5 件全合格写成：

```text
P(QQQQQ) = p^5
```

其中：

```text
p = 单件合格概率
```

如果 `p = 0.90`：

```text
P(QQQQQ) = 0.90^5 = 0.59049
```

如果 `p = 0.95`：

```text
P(QQQQQ) = 0.95^5 ≈ 0.77378
```

如果 `p = 0.98`：

```text
P(QQQQQ) = 0.98^5 ≈ 0.90392
```

请完成这句话：

```text
同样看到 QQQQQ，在 p=0.90、p=0.95、p=0.98 三个模型下，它的“常见程度” ________。
```

标准答案：

```text
不同
```

这就是概率模型的作用：

> 它不是只问“结果发生了吗”。  
> 它还问“在某个假设的生成过程里，这个结果有多常见”。

现在讨论独立性。

两个事件独立，意思不是“它们看起来没有关系”。数学表达是：

```text
P(A 且 B) = P(A) × P(B)
```

放到产品上：

```text
第1件是否合格，不改变第2件是否合格的概率。
```

但工业现场里，独立性经常需要被审查：

| 情况 | 独立性是否值得审查 | 原因 |
| --- | --- | --- |
| 连续来自同一卷材料 | 是 | 同一段材料的缺陷会成片出现 |
| 连续来自同一台设备 | 是 | 设备状态会持续影响后续产品 |
| 每件来自充分混合后的大批次 | 是 | 仍需说明抽样方式 |
| 每件来自不同供应商批次 | 是 | 不同来源引入分组差异 |

请写下：

```text
独立性不是默认真理，而是模型假设。
```

现在改写第三次判断：

```text
第三次判断：质量很好 / 不能判断 / 需要继续抽检
这次判断依赖的模型假设：
最需要核实的现场信息：
```

此刻你应该能说清：

```text
公式 p^5 很简单，真正关键的是 p 从哪里来，以及独立性是否成立。
```

## 看见期望

概率不只用来描述事件发生的机会，也用来描述长期平均结果。

定义一个变量：

```text
X = 单件产品是否合格

合格：X = 1
不合格：X = 0
```

如果单件合格概率为 `p`：

```text
P(X = 1) = p
P(X = 0) = 1 - p
```

那么单件产品的期望是：

```text
E(X) = 1 × p + 0 × (1 - p) = p
```

这句话不用先背公式。它的直觉是：

```text
如果一件产品合格记 1，不合格记 0，
那么长期平均分数就是合格率。
```

如果一批抽检 100 件，总合格件数记为 `S`：

```text
E(S) = 100p
```

请完成表格：

| 单件合格率 p | 100件中期望合格件数 |
| ---: | ---: |
| 80% |  |
| 90% |  |
| 95% |  |
| 98% |  |

标准答案：

| 单件合格率 p | 100件中期望合格件数 |
| ---: | ---: |
| 80% | 80 |
| 90% | 90 |
| 95% | 95 |
| 98% | 98 |

现在加一个小问题。

如果单件利润为 20 元，不合格品返工成本为 80 元。每件产品只有两种结果：

```text
合格：收益 +20 元
不合格：收益 -80 元
```

当 `p = 0.95` 时，单件期望收益是多少？

请写：

```text
E(收益) = 20 × 0.95 + (-80) × 0.05
```

标准计算：

```text
E(收益) = 19 - 4 = 15 元
```

当 `p = 0.80` 时：

```text
E(收益) = 20 × 0.80 + (-80) × 0.20 = 16 - 16 = 0 元
```

这和第 1 次的损失结构连起来了。

> 概率模型描述不确定性。  
> 期望把概率和后果放到同一个表达里。  
> 决策还要继续加入风险、约束和行动成本。

此刻你应该能说清：

```text
期望不是下一次结果，而是模型下长期平均结果的压缩表达。
```

## 划清边界

现在回到开头问题：

> 连续 5 件产品合格，是否说明这一批产品质量很好？

请用三层语言回答。

第一层，只说结果：

```text
这次抽检中，连续5件产品全部合格。
```

第二层，加上模型：

```text
如果单件合格率为 p，且每件结果相互独立，
那么5件全合格的概率是 p^5。
```

第三层，说清边界：

```text
仅凭5件全合格，不能直接确认整批产品质量很好；
还需要抽样方式、批量大小、历史缺陷率、缺陷聚集方式和行动损失。
```

请把自己的最终判断写成下面格式：

```text
数据结论：
模型假设：
边界说明：
下一步行动：
```

参考表达：

```text
数据结论：当前看到的5件样本全部合格。
模型假设：若单件合格概率为 p 且结果独立，则该结果出现概率为 p^5。
边界说明：5件全合格不能单独证明整批质量很好。
下一步行动：继续核实抽样方式、历史合格率、批内缺陷聚集和降低抽检后的损失。
```

现在请圈出最关键的一句：

```text
同一个结果，在不同模型下含义不同。
```

此刻你应该能说清：

```text
概率模型让结果有了参照系，但模型假设也限制了结论边界。
```

## 连接下一课

今天我们一直在问：

```text
看到连续5件合格，应该怎样解释？
```

但我们刻意回避了一个更重要的问题：

```text
如果我又知道这5件来自同一卷材料、同一台设备、同一个班次，
我讨论的对象是否已经改变？
```

这就是下一课的入口：条件概率。

先写下三个句子：

```text
没有额外信息时，我讨论的是整批产品。
知道“来自同一卷材料”以后，我讨论的是这卷材料对应的子总体。
知道“来自异常温度时段”以后，我讨论的是异常条件下的过程。
```

下一课的核心问题是：

```text
增加一条信息后，我们讨论的还是原来的总体吗？
```

今天不展开 `P(A|B)`。

只先留下这一句：

```text
条件不是公式里的竖线，条件会改变讨论对象。
```

此刻你应该能说清：

```text
概率模型回答“在某个模型下结果多常见”，条件概率回答“在新信息下对象是否改变”。
```

## 收束：用三句话带走今天

请在纸上写下三句话。

第一句：一句数据结论。

```text
连续5件产品全部合格。
```

第二句：一句模型结论。

```text
若单件合格概率为 p 且结果独立，则5件全合格的概率为 p^5。
```

第三句：一个下一课问题。

```text
如果这5件来自同一条件下的过程，我们讨论的还是整批产品吗？
```

现在把今天的核心判断写完整：

```text
连续5件合格会提高我的信心，
但它不能脱离抽样方式、模型假设和行动损失，
直接变成“整批质量很好”或“可以降低抽检”的结论。
```

## 本节板书总表

| 名称 | 直观含义 | 本节表达 |
| --- | --- | --- |
| 样本空间 | 模型考虑的全部结果 | `Ω = 所有5位Q/N序列` |
| 事件 | 我们关心的一组结果 | `A = {QQQQQ}` |
| 概率 | 事件在模型中的常见程度 | `P(A)` |
| 频率解释 | 多次重复下出现比例 | `100批中约59批全合格` |
| 可信程度 | 在现有信息下的信心表达 | “我对整批质量更放心，但仍需边界说明” |
| 独立性 | 前一件结果不改变后一件概率 | `P(A且B)=P(A)P(B)` |
| 5件全合格 | 独立模型下连续相乘 | `P(QQQQQ)=p^5` |
| 期望 | 模型下长期平均结果 | `E(X)=p`, `E(S)=100p` |
| 模型边界 | 结论依赖的条件 | 抽样方式、批内聚集、历史数据、行动损失 |
