## 开场：推断不是决策

前面几次课，我们一直在更新可信程度：

```text
报警后缺陷概率是多少？
试运行后设备可靠权重是多少？
新证据后哪个解释更可信？
```

今天要加上一句话：

```text
知道我相信什么，不等于知道我该做什么。
```

请写下今天的课堂契约：

```text
推断回答可信程度，决策回答在损失结构下选择行动。
```

今天的核心句是：

```text
同一个后验概率，在不同损失结构下会对应不同决策。
```

此刻你应该能说清：

```text
概率进入决策时，必须和后果成本放在一起。
```

## 先做判断

某关键设备报警。根据前面模型更新：

```text
报警后设备真实异常概率为 70%。
```

现在有两个行动：

```text
继续运行
立即停机检查
```

请先写：

```text
第一次判断：继续运行 / 停机检查 / 需要更多信息
理由：
我最需要知道的成本：
```

停 1 分钟。

现在回答：

1. 70% 对你来说高还是低？
2. 如果继续运行导致事故，损失是多少？
3. 如果停机检查但设备没异常，损失是多少？

请写下：

```text
没有损失结构，70%不能直接转成行动。
```

此刻你应该能说清：

```text
行动阈值不是固定概率，而是由错判代价决定。
```

## 加入损失

设定损失：

```text
如果设备异常但继续运行：损失100万元。
如果设备正常但停机检查：损失10万元。
如果设备异常且停机检查：损失20万元。
如果设备正常且继续运行：损失0万元。
```

整理成表：

| 真实状态 | 继续运行损失 | 停机检查损失 |
| --- | ---: | ---: |
| 异常 | 100万元 | 20万元 |
| 正常 | 0万元 | 10万元 |

现在后验为：

```text
P(异常|报警)=70%
P(正常|报警)=30%
```

请先不计算，判断：

```text
哪种行动更合理：
理由：
```

此刻你应该能说清：

```text
决策表把真实状态、行动和后果放在同一张表里。
```

## 算期望损失

期望损失的基本形式：

```text
期望损失 = 发生概率 × 后果成本，再对所有状态求和
```

继续运行：

```text
E(继续运行损失)
= P(异常) × 100 + P(正常) × 0
= 0.70 × 100 + 0.30 × 0
= 70 万元
```

停机检查：

```text
E(停机检查损失)
= P(异常) × 20 + P(正常) × 10
= 0.70 × 20 + 0.30 × 10
= 14 + 3
= 17 万元
```

所以在这个损失结构下：

```text
停机检查的期望损失更低。
```

请改写判断：

```text
第二次判断：
我改变判断的原因：
```

此刻你应该能说清：

```text
行动选择来自期望损失比较，而不是只看异常概率大小。
```

## 找到阈值

现在不固定 70%。设：

```text
q = P(异常|报警)
```

继续运行期望损失：

```text
100q
```

停机检查期望损失：

```text
20q + 10(1-q)
= 10 + 10q
```

停机检查更优时：

```text
10 + 10q < 100q
10 < 90q
q > 1/9 ≈ 11.1%
```

这就是行动阈值。

请写：

```text
只要异常概率超过约11.1%，停机检查的期望损失就低于继续运行。
```

现在回答：

```text
为什么阈值远低于50%：
```

参考表达：

```text
因为异常但继续运行的损失远大于正常但停机检查的损失。
```

此刻你应该能说清：

```text
高后果低概率事件，也会触发行动。
```

## 加入信息价值

现在有第三个行动：

```text
追加复检，成本2万元。
复检能把异常判断进一步澄清。
```

管理者问：

```text
复检值不值得？
```

先不算复杂公式，只写决策逻辑。

复检有价值，必须满足：

```text
复检带来的期望损失下降 > 复检成本
```

请填表：

| 复检结果 | 后续行动 | 复检价值来源 |
| --- | --- | --- |
| 强烈支持异常 |  |  |
| 强烈支持正常 |  |  |
| 结果仍不清楚 |  |  |

参考表达：

| 复检结果 | 后续行动 | 复检价值来源 |
| --- | --- | --- |
| 强烈支持异常 | 停机维修 | 避免事故损失 |
| 强烈支持正常 | 继续运行 | 避免无效停机 |
| 结果仍不清楚 | 按原阈值决策 | 信息价值有限 |

此刻你应该能说清：

```text
信息的价值来自它能改变行动，而不是来自它让报告更完整。
```

## 迁移到质量和维修

现在换三个场景。任选一个，写出决策表。

### 场景一：产品返工

```text
某批产品有缺陷概率为30%。
放行后被客户发现，损失50万元。
返工成本8万元。
```

请写：

```text
行动：
真实状态：
关键损失：
```

参考表达：

```text
行动：放行 / 返工。
真实状态：有缺陷 / 无缺陷。
关键损失：有缺陷放行的客户损失，正常产品返工的无效成本。
```

### 场景二：设备预防维修

```text
设备未来一周失效概率为20%。
失效停线损失80万元。
预防维修成本12万元。
```

参考表达：

```text
行动：继续运行 / 预防维修。
真实状态：本周失效 / 本周不失效。
关键损失：失效停线损失，预防维修成本。
```

### 场景三：增加抽检

```text
供应商批次异常概率为15%。
增加抽检成本3万元。
漏放异常批次损失40万元。
```

参考表达：

```text
行动：常规抽检 / 增加抽检。
真实状态：批次异常 / 批次正常。
关键损失：漏放异常批次，增加抽检成本。
```

此刻你应该能说清：

```text
推断结果进入工业现场时，最先要变成决策表。
```

## 划清边界

期望损失能帮助选择行动，但它不是全部。

还要说明：

| 因素 | 为什么重要 |
| --- | --- |
| 风险偏好 | 同一期望下，有人更厌恶极端损失 |
| 资金约束 | 最优行动也要满足现金和产能限制 |
| 法规要求 | 某些风险不能只用成本权衡 |
| 声誉损失 | 金额难以准确量化 |
| 多目标冲突 | 质量、交付、成本同时存在 |

请写下边界句：

```text
期望损失是决策骨架，不是替代管理判断的全部系统。
```

完成最终输出：

```text
推断结论：
损失结构：
行动阈值：
边界说明：
```

此刻你应该能说清：

```text
统计推断进入行动前，必须经过损失结构和约束条件。
```

## 连接下一课

到这里，我们知道怎样从后验走向行动。

下一课回到一个更基础的问题：

```text
这些后验、概率和参数，怎样由样本支持总体？
```

我们要比较两种语言：

```text
频率学派：如果重复抽样，程序长期表现怎样？
贝叶斯语言：在当前数据下，参数或假设可信程度怎样？
```

此刻你应该能说清：

```text
第8次解决“如何行动”；第9次追问“样本怎样支持总体结论”。
```

## 收束：用三句话带走今天

第一句：一句推断结论。

```text
报警后设备异常概率为70%。
```

第二句：一句决策结论。

```text
在当前损失表下，停机检查期望损失为17万元，低于继续运行的70万元。
```

第三句：一个下一课问题。

```text
这些概率和参数怎样由样本数据支持？
```

## 本节板书总表

| 名称 | 直观含义 | 本节表达 |
| --- | --- | --- |
| 推断 | 我相信什么 | `P(异常|报警)=70%` |
| 行动 | 我要做什么 | 继续运行或停机检查 |
| 损失表 | 状态和行动的后果 | 异常继续运行损失100万元 |
| 期望损失 | 概率加权后的平均损失 | `E(L)=Σ概率×损失` |
| 行动阈值 | 换行动的概率临界点 | `q>11.1%` |
| 信息价值 | 新信息降低期望损失 | 复检改变后续行动 |
| 决策边界 | 期望损失之外的约束 | 法规、声誉、资金、风险偏好 |
