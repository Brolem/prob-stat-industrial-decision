## 开场：综合判断不是套单个公式

今天不引入新公式。

我们只做一件事：

```text
把前10次课的判断结构放进一个工业过程问题。
```

请写下今天的课堂契约：

```text
完整工业判断 = 定义问题 + 识别证据 + 更新判断 + 区分因果 + 连接行动 + 说明边界。
```

今天的核心句是：

```text
统计工具不是孤立技巧，而是服务于过程判断和工业行动的语言系统。
```

此刻你应该能说清：

```text
最后一课的目标是整合，不是学习更多术语。
```

## 先做判断

某材料厂生产一种关键材料，核心性能指标为强度 `S`。

历史稳定生产时：

```text
S 的均值约为 100
标准差约为 2
客户规格要求：S ≥ 96
```

最近 8 批抽检均值如下：

| 批次 | 当前抽检均值 |
| ---: | ---: |
| 1 | 99.1 |
| 2 | 98.7 |
| 3 | 98.4 |
| 4 | 97.9 |
| 5 | 97.6 |
| 6 | 97.2 |
| 7 | 96.9 |
| 8 | 96.5 |

管理者问：

```text
是否继续生产、增加检测、调整参数，还是停机？
```

请先写：

```text
第一次判断：继续 / 增加检测 / 调整参数 / 停机
理由：
我最担心的风险：
```

停 1 分钟。

现在回答：

1. 这些批次是否都还在规格内？
2. 均值是否出现连续下降？
3. 在规格内是否就代表过程没有问题？

请写下：

```text
符合规格不等于过程稳定。
```

此刻你应该能说清：

```text
工业判断要同时看规格、趋势、波动和后果。
```

## 打开数据

把历史基准和当前数据放在一起。

| 项目 | 历史稳定过程 | 当前8批 |
| --- | ---: | ---: |
| 均值中心 | 100 | 从99.1降到96.5 |
| 典型波动 | 标准差约2 | 连续下滑 |
| 规格下限 | 96 | 第8批接近下限 |
| 主要风险 | 偶发低值 | 趋势性靠近下限 |

请把观察写成三句话：

```text
水平：
趋势：
风险：
```

参考表达：

```text
水平：当前批次均值仍大多高于规格下限96。
趋势：8批数据呈连续下降。
风险：若趋势延续，后续批次触碰或低于规格下限的机会升高。
```

现在改写判断：

```text
第二次判断：
我改变判断的原因：
```

此刻你应该能说清：

```text
当前数据支持“过程正在向低强度方向移动”的警惕判断。
```

## 更新判断

现在加入新证据。

在线传感器报警：

```text
过去类似报警后，有70%的案例在后续24小时内出现强度继续下降。
误报警率为20%。
```

破坏性试验补充结果：

```text
第8批额外抽取5件做破坏性试验，结果有2件低于96。
```

请先写：

```text
新证据1：
新证据2：
它们怎样改变判断：
```

参考表达：

```text
新证据1：在线传感器报警，提示后续下降风险升高。
新证据2：破坏性试验中5件有2件低于规格下限。
它们把判断从“趋势警惕”推向“质量风险已经进入行动区”。
```

现在把证据分层：

| 证据 | 支持什么 | 不支持什么 |
| --- | --- | --- |
| 8批均值下降 | 过程中心下移 | 不能单独说明原因 |
| 传感器报警 | 后续下降风险升高 | 不能单独确认失控 |
| 破坏性试验2/5低于规格 | 当前批次存在低值风险 | 样本量小，需继续确认比例 |

此刻你应该能说清：

```text
不同证据支持不同层次的结论，不能混成一句“过程坏了”。
```

## 区分异常与原因

现在问原因。

现场同时发生三件事：

```text
原材料供应商更换。
设备温控系统刚维护。
夜班操作人员换班。
```

请回答：

```text
哪一个是原因：
```

不要急着选。先列替代解释。

| 解释 | 支持证据 | 还缺什么 |
| --- | --- | --- |
| 原材料变化 | 更换供应商后强度下降 | 新旧材料同期对照 |
| 温控维护问题 | 维护后过程趋势变化 | 温度记录和校准验证 |
| 班组变化 | 夜班换班 | 分班次质量数据 |
| 随机波动 | 小样本下会波动 | 趋势持续性和更多批次 |

请写下：

```text
异常信号不等于原因已经确认。
```

此刻你应该能说清：

```text
过程变化是推断问题，原因归属是因果问题。
```

## 加入损失

现在加入行动成本。

| 行动 | 若过程继续下滑 | 若过程只是短期波动 |
| --- | ---: | ---: |
| 继续生产 | 客诉和召回损失200万元 | 无额外成本 |
| 增加检测 | 发现风险，成本15万元 | 成本15万元 |
| 调整参数 | 降低风险，成本30万元 | 成本30万元且影响产能 |
| 停机排查 | 避免大损失，成本80万元 | 成本80万元 |

请先写直觉：

```text
第三次判断：继续 / 增加检测 / 调整参数 / 停机
理由：
```

现在给一个当前判断：

```text
过程继续下滑的可信程度为60%。
```

粗算期望损失。

继续生产：

```text
0.60 × 200 + 0.40 × 0 = 120万元
```

增加检测：

```text
固定成本15万元，同时保留进一步行动机会。
```

停机排查：

```text
固定成本80万元，直接控制大损失风险。
```

请回答：

```text
为什么“增加检测”在这里是自然的中间行动：
```

参考表达：

```text
因为它成本低于停机，且能快速获得更多证据，帮助决定是否调参或停机。
```

此刻你应该能说清：

```text
行动不只有继续和停机，工业决策常包含分阶段升级。
```

## 形成行动条件

把行动写成条件，而不是一句口号。

建议行动框架：

```text
立即增加检测。
同时核查温控记录、原材料批次和分班次数据。
若下一批均值低于96.5或破坏性试验再出现低于96，则调整参数并隔离相关批次。
若连续两批低于规格或传感器持续报警，则停机排查。
```

请把这段拆成表：

| 条件 | 行动 |
| --- | --- |
| 当前证据 |  |
| 下一批均值低于96.5 |  |
| 破坏性试验再低于96 |  |
| 连续两批低于规格 |  |
| 传感器持续报警 |  |

参考表达：

| 条件 | 行动 |
| --- | --- |
| 当前证据 | 增加检测并核查关键记录 |
| 下一批均值低于96.5 | 调整参数 |
| 破坏性试验再低于96 | 隔离相关批次 |
| 连续两批低于规格 | 停机排查 |
| 传感器持续报警 | 升级为设备和工艺联合排查 |

此刻你应该能说清：

```text
好的决策建议要写清触发条件，而不是只写最终动作。
```

## 写综合报告

请用下面模板写一段完整报告。

```text
问题定义：
当前证据：
概率/统计判断：
因果边界：
损失结构：
行动建议：
下一步数据：
```

参考报告：

```text
问题定义：判断材料强度过程是否出现下移，并决定继续生产、加检、调参或停机。
当前证据：8批均值连续从99.1降到96.5，传感器报警，第8批破坏性试验5件中2件低于96。
概率/统计判断：数据支持过程向低强度方向移动的警惕判断，且当前批次低值风险上升。
因果边界：现有证据不能确认原因来自原材料、温控维护或班组变化，需要对照和分层数据。
损失结构：继续生产在过程继续下滑时有高额客诉和召回风险，增加检测成本较低且能获得新证据。
行动建议：立即增加检测，隔离第8批，核查温控、原材料和分班次记录；按后续阈值升级调参或停机。
下一步数据：下一批均值、破坏性试验结果、温控记录、供应商批次对照、班组分层质量数据。
```

此刻你应该能说清：

```text
综合报告必须同时给结论、边界和行动条件。
```

## 划清边界

最后一课最容易犯的错误，是把所有证据压成一句过强结论。

请把边界逐层写开：

| 层次 | 当前能说 | 当前不能说 |
| --- | --- | --- |
| 数据描述 | 8批均值连续下降 | 不能说长期趋势已经完全确定 |
| 概率判断 | 后续低值风险上升 | 不能说下一批一定低于规格 |
| 样本推断 | 破坏性试验显示当前批次有低值风险 | 不能把2/5直接当作总体比例 |
| 因果判断 | 原材料、温控、班组都值得审查 | 不能直接确认单一原因 |
| 决策建议 | 增加检测和隔离风险批次有行动价值 | 不能用一个固定规则替代现场约束 |

请写下边界句：

```text
完整工业判断不是把结论说满，而是把证据强度、因果边界和行动条件说清。
```

此刻你应该能说清：

```text
越是综合判断，越要把“能说什么”和“不能说什么”分开。
```

## 收束：一个完整工业判断模板

最后写下全课模板。

```text
1. 我们要判断什么？
2. 当前数据是什么？
3. 数据支持什么？
4. 数据不支持什么？
5. 需要哪个概率或统计模型？
6. 新证据如何更新判断？
7. 有哪些替代解释和因果边界？
8. 错判、漏判和行动成本是什么？
9. 当前建议行动是什么？
10. 触发升级行动的条件是什么？
```

把今天案例浓缩成三句话：

```text
数据结论：8批强度均值连续下降，且第8批破坏性试验出现低于规格的样本。
边界说明：现有证据支持过程风险上升，但不能直接确认原材料、温控或班组中的具体原因。
行动建议：立即增加检测并隔离风险批次，同时按后续阈值决定调参或停机。
```

---

## 本节板书总表

| 模块 | 关键问题 | 本节表达 |
| --- | --- | --- |
| 问题定义 | 判断对象是什么 | 强度过程是否下移 |
| 数据描述 | 看到什么 | 8批均值连续下降 |
| 概率更新 | 新证据如何改变判断 | 传感器报警和破坏性试验提高风险判断 |
| 分布判断 | 过程取值怎样变化 | 强度中心靠近规格下限 |
| 样本边界 | 样本支持什么 | 5件破坏性试验证据强但样本小 |
| 因果边界 | 原因是否确认 | 原材料、温控、班组都需审查 |
| 损失结构 | 错判代价是什么 | 继续下滑损失高，加检成本低 |
| 行动条件 | 何时升级 | 均值阈值、规格低值、持续报警 |
| 综合报告 | 如何表达 | 问题、证据、边界、行动、下一步 |
