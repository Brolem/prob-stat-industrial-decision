## 开场：先验不是随意猜

上节课我们连续更新：

```text
先验 -> 证据 -> 后验 -> 新证据 -> 新后验
```

今天只问一个基础问题：

```text
先验从哪里来？
```

请写下今天的课堂契约：

```text
先验必须有来源，模型必须有边界，争议必须能被说清。
```

今天的核心句是：

```text
先验不是随意判断，模型正确也不等于现实正确。
```

此刻你应该能说清：

```text
贝叶斯方法的争议不在公式本身，而在输入、模型和解释边界。
```

## 先做判断

某工厂引入一台新设备。没有这台设备的长期数据，但有四类信息：

```text
同型号旧设备历史数据
供应商可靠性报告
工程师对结构的物理判断
前3天试运行记录
```

管理者问：

```text
我们能否给这台设备建立可靠性先验？
```

请先写：

```text
第一次判断：可以 / 不可以 / 可以但要注明来源
我最信任的信息：
我最不信任的信息：
```

停 1 分钟。

现在回答：

1. 历史数据和专家经验哪个更可靠？
2. 供应商报告是否可以直接当作先验？
3. 新设备与旧设备差异会怎样影响先验？

请写下：

```text
先验不是一个数字，而是一组信息来源被压缩后的判断。
```

此刻你应该能说清：

```text
先验的质量取决于来源、适用性和公开程度。
```

## 列出来源

先验常见来源有四类。

| 来源 | 优点 | 风险 |
| --- | --- | --- |
| 历史数据 | 有实际记录 | 工况、材料、人员已变化 |
| 同类设备 | 有可比对象 | 型号差异被低估 |
| 物理机理 | 能解释原因 | 机理简化过度 |
| 专家经验 | 能利用隐性知识 | 主观偏差和过度自信 |

请把新设备问题中的四类信息放入表格：

| 信息 | 可用性 | 风险 |
| --- | --- | --- |
| 同型号旧设备历史数据 |  |  |
| 供应商可靠性报告 |  |  |
| 工程师物理判断 |  |  |
| 前3天试运行记录 |  |  |

参考表达：

| 信息 | 可用性 | 风险 |
| --- | --- | --- |
| 同型号旧设备历史数据 | 可作为先验起点 | 旧工况与新工况不同 |
| 供应商可靠性报告 | 可作为外部证据 | 测试条件与现场不同 |
| 工程师物理判断 | 可约束模型形状 | 机理遗漏 |
| 前3天试运行记录 | 可作为新证据 | 样本短、工况窄 |

此刻你应该能说清：

```text
先验可以来自数据，也可以来自机理和经验，但必须说明来源和风险。
```

## 比较先验

我们回到设备可靠性问题。

有两种解释：

```text
H1：设备可靠
H2：设备一般
```

证据仍然是：

```text
E = 5次试运行全部成功
P(E|H1)=0.774
P(E|H2)=0.328
```

现在比较三组先验：

| 先验方案 | `P(H1)` | `P(H2)` | 含义 |
| --- | ---: | ---: | --- |
| 保守先验 | 30% | 70% | 新设备风险高 |
| 中性先验 | 50% | 50% | 两种解释同等权重 |
| 乐观先验 | 70% | 30% | 同类设备表现好 |

请计算更新后 `P(H1|E)`。

标准结果：

| 先验方案 | 更新后 `P(H1|E)` |
| --- | ---: |
| 保守先验 | `0.30×0.774/(0.30×0.774+0.70×0.328)≈50.3%` |
| 中性先验 | `≈70.2%` |
| 乐观先验 | `≈84.6%` |

请回答：

```text
同一个证据，为什么后验不同：
```

参考表达：

```text
因为证据是在不同先验起点上更新的。
```

此刻你应该能说清：

```text
同一份数据不会消除所有先验差异。
```

## 做敏感性分析

敏感性分析就是问：

```text
如果先验或模型输入改变，结论是否仍然支撑同一行动？
```

请看表：

| 先验方案 | 后验 `P(H1|E)` | 若行动阈值为80%，是否投产 |
| --- | ---: | --- |
| 保守先验 | 50.3% | 否 |
| 中性先验 | 70.2% | 否 |
| 乐观先验 | 84.6% | 是 |

请写判断：

```text
这个决策对先验是否敏感：
原因：
```

标准表达：

```text
敏感。因为不同合理先验会导致是否投产的行动结论不同。
```

现在把行动阈值改成 50%。

```text
若只要求H1后验超过50%，三种先验下是否都投产：
```

标准回答：

```text
三种方案都刚好或明显超过50%，行动结论更稳定。
```

此刻你应该能说清：

```text
敏感性分析不是为了回避先验，而是为了公开结论对先验的依赖。
```

## 审查模型

公式算对了，模型仍要审查。

请看这个可靠性模型：

```text
每次试运行成功率固定。
每次试运行相互独立。
试运行工况代表正式生产工况。
成功和失败定义清晰。
```

逐项提问：

| 假设 | 审查问题 |
| --- | --- |
| 成功率固定 | 设备是否会随磨合、温度、负载改变 |
| 相互独立 | 上一次故障是否影响下一次试验 |
| 工况代表 | 试运行是否覆盖正式生产负荷 |
| 定义清晰 | 成功是否只指能启动，还是满足产能和质量 |

请写下：

```text
模型正确计算，只说明在这些假设下结论成立。
```

此刻你应该能说清：

```text
模型假设是结论的一部分，不能藏在公式后面。
```

## 识别偏差

先验和模型常见偏差：

| 偏差 | 表现 | 影响 |
| --- | --- | --- |
| 样本偏差 | 只记录顺利试运行 | 高估可靠性 |
| 选择偏差 | 只选好批次做测试 | 高估过程能力 |
| 幸存者偏差 | 只分析没有报废的设备 | 低估早期失效 |
| 测量偏差 | 传感器长期偏高或偏低 | 错估质量指标 |
| 报告偏差 | 失败记录不完整 | 后验过度乐观 |

请选一个偏差，写成工业句子：

```text
偏差名称：
在本项目中的表现：
它会怎样改变判断：
```

参考表达：

```text
偏差名称：报告偏差。
在本项目中的表现：供应商报告中只列合格测试，不列失败测试。
它会怎样改变判断：先验可靠性被抬高，后验投产信心过强。
```

此刻你应该能说清：

```text
争议不是统计方法的失败，而是证据来源和模型边界需要被公开。
```

## 迁移到工业评审

现在模拟一次投产评审。请用四句话汇报。

```text
我采用的先验来自：
我采用的模型假设是：
敏感性分析显示：
我建议的下一步证据是：
```

参考表达：

```text
我采用的先验来自同型号旧设备历史数据和供应商测试报告。
我采用的模型假设是试运行成功率固定且每次试验近似独立。
敏感性分析显示在保守和中性先验下尚未达到80%投产阈值。
我建议继续做覆盖高负载工况的试运行，并完整记录失败。
```

此刻你应该能说清：

```text
好的统计汇报不是只给一个数，而是同时交代来源、假设、敏感性和下一步证据。
```

## 划清边界

先验不是随意，模型也不是现实。

请把今天的边界句写完整：

```text
我可以使用先验，但必须说明：
我可以使用模型，但必须说明：
我可以给出后验，但必须说明：
```

参考表达：

```text
我可以使用先验，但必须说明来源和适用范围。
我可以使用模型，但必须说明关键假设。
我可以给出后验，但必须说明它对先验和模型输入的敏感性。
```

此刻你应该能说清：

```text
可信的贝叶斯分析不是消除争议，而是把争议放到可讨论的位置。
```

## 连接下一课

到目前为止，我们一直在问：

```text
我相信什么？
```

下一课要问：

```text
在损失结构下，我做什么？
```

即使后验达到 70%，行动也不自动确定。

请写下下一课问题：

```text
可信程度怎样转化为行动阈值？
```

此刻你应该能说清：

```text
推断回答相信什么，决策回答在代价约束下做什么。
```

## 收束：用三句话带走今天

第一句：一句先验来源说明。

```text
本分析的先验来自同类设备历史数据、供应商报告和工程机理判断。
```

第二句：一句边界说明。

```text
后验结论对先验和试运行模型敏感，不能脱离敏感性分析解释。
```

第三句：一个下一课问题。

```text
当后验可信程度已经得到，怎样用损失结构决定是否行动？
```

## 本节板书总表

| 名称 | 直观含义 | 本节表达 |
| --- | --- | --- |
| 先验来源 | 更新前信息从哪里来 | 历史、同类、机理、专家 |
| 弱信息先验 | 承认信息有限的起点 | 中性或宽松设定 |
| 敏感性分析 | 改变输入看结论是否稳 | 保守、中性、乐观先验 |
| 模型假设 | 计算成立的条件 | 固定成功率、独立、工况代表 |
| 样本偏差 | 数据来源不代表目标对象 | 只记录顺利试验 |
| 模型不确定性 | 模型形式本身存在争议 | 成功率是否固定 |
| 决策入口 | 推断转行动的桥 | 行动阈值和损失结构 |
