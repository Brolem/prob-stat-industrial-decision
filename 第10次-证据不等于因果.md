## 开场：证据不自动说明原因

前面课程一直在更新判断。

今天要加一个限制：

```text
证据支持差异，不等于证据已经支持因果。
```

请写下今天的课堂契约：

```text
看到变化后，先列出替代解释，再谈因果。
```

今天的核心句是：

```text
因果结论的强度，取决于证据来源和比较设计。
```

此刻你应该能说清：

```text
概率更新不能替代实验设计。
```

## 先做判断

某工厂把工艺参数 A 从 60 调到 70。调整后 7 天，不合格率从 5% 降到 3%。

管理者问：

```text
是否可以说参数A调整导致质量改善？
```

请先写：

```text
第一次判断：可以 / 不可以 / 需要更多设计信息
理由：
我最担心的替代解释：
```

停 1 分钟。

现在回答：

1. 质量改善这个现象是否存在？
2. 改善是否一定来自参数 A？
3. 还有哪些因素同时改变？

请写下：

```text
事实变化：不合格率下降。
因果解释：参数A导致下降。
这两句话不是同一件事。
```

此刻你应该能说清：

```text
先承认证据，再审查因果解释。
```

## 列出替代解释

请列出除了参数 A 以外，导致不合格率下降的因素。

参考表：

| 替代解释 | 怎样影响结论 |
| --- | --- |
| 原材料批次更好 | 把材料改善误归因给工艺 |
| 操作人员更熟练 | 把人员差异误归因给参数 |
| 设备刚维护 | 把设备状态误归因给工艺 |
| 检测标准改变 | 把测量差异误归因给质量 |
| 订单难度降低 | 把产品组合变化误归因给工艺 |

这些因素叫混杂变量：

```text
混杂变量：同时影响处理选择和结果的因素。
```

在本例中：

```text
处理：参数A从60调到70
结果：不合格率
混杂变量：材料、人员、设备、检测、订单组合
```

请写下：

```text
如果混杂变量没有控制，因果结论就不稳固。
```

此刻你应该能说清：

```text
混杂会让两个变量看起来有关，但真实原因来自第三类因素。
```

## 设计对照

要更接近因果结论，需要公平比较。

请看三种设计：

| 设计 | 做法 | 因果强度 |
| --- | --- | --- |
| 前后对比 | 改工艺前后比较 | 弱，容易受时间变化影响 |
| 同期对照 | 同期一条线用旧参数，一条线用新参数 | 中，仍要审查两条线差异 |
| 随机分配 | 同类产品随机分到旧参数和新参数 | 强，能打散已知和未知差异 |

请回答：

```text
为什么前后对比因果强度弱：
为什么同期对照更好：
为什么随机分配更强：
```

参考表达：

```text
前后对比把时间变化混在一起。
同期对照减少时间变化影响。
随机分配能让两组在已知和未知因素上更可比。
```

此刻你应该能说清：

```text
因果设计的目标，是让处理组和对照组除了处理之外尽量相同。
```

## 理解随机化

随机化不是为了制造混乱，而是为了打散差异。

假设有 100 个生产批次。我们随机分成两组：

```text
50个批次使用旧参数A=60
50个批次使用新参数A=70
```

如果随机分配执行得好：

```text
材料、人员、订单难度、设备时段等因素会在两组中相对平衡。
```

请写下：

```text
随机化让处理组和对照组在处理前更可比。
```

但随机化之后仍要检查：

| 检查项 | 作用 |
| --- | --- |
| 是否真正随机 | 防止人为选择 |
| 样本量是否足够 | 防止偶然不平衡 |
| 测量是否一致 | 防止检测偏差 |
| 是否有中途退出 | 防止选择偏差 |
| 是否能外推 | 防止只适用于特殊条件 |

此刻你应该能说清：

```text
随机化提高因果识别强度，但仍要看执行和外推边界。
```

## 迁移到工业实验

现在换三个工业问题。任选一个，设计对照。

### 情境一：新原材料

```text
更换新原材料后，强度提高。
```

请写：

```text
处理：
结果：
混杂：
对照设计：
```

参考表达：

```text
处理：使用新原材料。
结果：材料强度。
混杂：供应批次、设备状态、固化温度、操作人员。
对照设计：同一时间内随机安排批次使用新旧原材料，保持其他条件一致。
```

### 情境二：设备参数

```text
提高压力后，表面缺陷减少。
```

参考表达：

```text
处理：压力设定提高。
结果：单位面积表面缺陷数。
混杂：材料湿度、模具状态、检测人员。
对照设计：同类批次随机分配到旧压力和新压力，并统一检测标准。
```

### 情境三：培训制度

```text
班组培训后，返工率下降。
```

参考表达：

```text
处理：接受培训。
结果：返工率。
混杂：班组经验、订单复杂度、设备差异。
对照设计：相似班组随机或分层安排培训，并跟踪同期返工率。
```

此刻你应该能说清：

```text
工业因果问题要先写处理、结果、混杂和对照。
```

## 识别证据等级

不同证据支持因果的强度不同。

| 证据来源 | 能说明什么 | 不能直接说明什么 |
| --- | --- | --- |
| 单次前后对比 | 改变前后结果不同 | 不能排除时间和环境变化 |
| 多批次观察数据 | 变量间有关联 | 不能完全排除混杂 |
| 匹配或分层比较 | 在已知因素上更公平 | 不能排除未知混杂 |
| 随机对照实验 | 强化因果识别 | 外推仍需审查 |
| 机理和实验一致 | 提升解释可信度 | 仍需确认边界条件 |

请把下面结论改写得更严谨：

原句：

```text
参数A提高导致不合格率下降。
```

更严谨表达：

```text
在当前观察数据中，参数A提高后不合格率下降；
若要支持因果结论，需要控制材料、设备、人员和检测标准等混杂因素。
```

此刻你应该能说清：

```text
严谨表达不会把观察差异直接写成因果结论。
```

## 划清边界

请把三句话分开。

```text
相关结论：A变化时，B也变化。
预测结论：看到A，可以帮助预测B。
因果结论：改变A，会导致B变化。
```

这三句话强度不同。

请写下边界句：

```text
能预测，不等于能干预；能更新概率，不等于证明因果。
```

完成最终输出：

```text
证据结论：
替代解释：
因果设计：
边界说明：
```

此刻你应该能说清：

```text
因果判断要把证据、替代解释和设计强度同时说出来。
```

## 收束：用三句话带走今天

第一句：一句证据结论。

```text
参数A调整后，不合格率从5%降到3%。
```

第二句：一句因果边界。

```text
仅凭前后对比，不能排除材料、设备、人员、检测和订单组合等混杂因素。
```

第三句：一个综合问题。

```text
当过程指标波动、报警和损失同时出现时，如何做完整工业判断？
```

## 本节板书总表

| 名称 | 直观含义 | 本节表达 |
| --- | --- | --- |
| 相关 | 两个变量一起变化 | 参数变，不合格率也变 |
| 因果 | 改变A导致B变化 | 参数改变导致质量改善 |
| 混杂 | 第三因素影响比较 | 材料、设备、人员 |
| 对照组 | 没接受处理的比较对象 | 旧参数组 |
| 随机化 | 打散已知和未知差异 | 批次随机分配 |
| 选择偏差 | 进入组别不公平 | 只给好批次用新工艺 |
| 外推 | 从实验条件推广到现场 | 其他产品、班次、季节 |
| 因果边界 | 结论成立条件 | 设计、执行、测量和外推 |
