## 开场：分布不是公式清单

前四次课我们一直在讨论事件：

```text
合格 / 不合格
报警 / 不报警
有缺陷 / 无缺陷
```

今天我们把问题换成：

```text
一批里有几个不合格？
一个零件的测量误差是多少？
一个设备还能运行多久？
```

请写下今天的课堂契约：

```text
分布不是公式清单，分布是对数据生成机制的模型。
```

今天的核心动作是：

```text
先定义随机变量 -> 再列出取值 -> 再描述每个取值的机会 -> 最后说明模型边界
```

此刻你应该能说清：

```text
事件只问是否发生，随机变量把结果变成可分析的数值。
```

## 先做判断

某生产线单件不合格率为 5%。现在随机抽检 10 件。

管理者问：

```text
这10件里会有几个不合格？
```

请不要计算，先写直觉判断：

```text
第一次判断：最常见的不合格件数：
我最担心的不合格件数：
我需要知道的模型假设：
```

停 1 分钟。

现在回答：

1. 你会不会直接说“10 × 5% = 0.5 件”？
2. 0.5 件是不是真实会出现的件数？
3. 10 件里出现 0 件不合格，和出现 1 件不合格，哪个更常见？

请写下：

```text
期望可以是0.5件，但实际取值只能是0、1、2、...、10件。
```

此刻你应该能说清：

```text
期望是长期平均，不是下一批的实际结果。
```

## 定义变量

定义随机变量：

```text
X = 10件产品中的不合格件数
```

那么 `X` 的取值是：

```text
0, 1, 2, 3, ..., 10
```

如果每件产品不合格概率为 `p = 0.05`，且每件结果相互独立，那么：

```text
X 服从二项分布
X ~ Binomial(n=10, p=0.05)
```

先不要背公式。先用一句话解释：

```text
二项分布描述固定次数独立试验中，某类结果出现的次数。
```

请把下面三个例子翻译成随机变量。

| 场景 | 随机变量 |
| --- | --- |
| 20件里不合格件数 |  |
| 100个焊点里开裂焊点数 |  |
| 30天里设备故障天数 |  |

参考表达：

| 场景 | 随机变量 |
| --- | --- |
| 20件里不合格件数 | `X = 20件中的不合格件数` |
| 100个焊点里开裂焊点数 | `Y = 100个焊点中的开裂数` |
| 30天里设备故障天数 | `Z = 30天中的故障天数` |

此刻你应该能说清：

```text
随机变量先把现实结果变成一个有单位、有取值范围的数字。
```

## 列出分布

对于 `X = 10件中的不合格件数`，模型给出每个取值的概率。

在 `n=10, p=0.05` 下，常见取值如下：

| 不合格件数 X | 概率 |
| ---: | ---: |
| 0 | 59.9% |
| 1 | 31.5% |
| 2 | 7.5% |
| 3 | 1.0% |
| 4及以上 | 0.1% |

请先看表回答：

```text
最常见的结果：
出现1件不合格是否罕见：
出现3件不合格是否值得关注：
```

标准回答：

```text
最常见的结果是0件不合格。
出现1件不合格不罕见。
出现3件不合格在该模型下已经少见，值得关注。
```

现在写下公式：

```text
P(X=k) = C(n,k) p^k (1-p)^(n-k)
```

在这里：

```text
n = 10
p = 0.05
k = 不合格件数
```

请计算 `X=0` 的概率：

```text
P(X=0) = 0.95^10 ≈ 59.9%
```

此刻你应该能说清：

```text
分布不是一个数，而是每个取值及其概率的完整表述。
```

## 看见参数

现在改变 `p`。

| 模型 | 不合格率 p | `P(X=0)` | `P(X≥2)` |
| --- | ---: | ---: | ---: |
| 过程较好 | 1% | 90.4% | 0.4% |
| 当前假设 | 5% | 59.9% | 8.6% |
| 过程变差 | 10% | 34.9% | 26.4% |

请回答：

```text
p 从1%到10%，分布发生了什么变化：
哪个事件对过程变差更敏感：X=0 还是 X≥2：
```

参考回答：

```text
p 升高时，0件不合格的概率下降，2件及以上不合格的概率上升。
X≥2 更适合提醒过程变差，因为它在好过程下很少见，在差过程下明显增加。
```

现在改变 `n`。

```text
同样 p=5%，抽检10件和抽检100件，看到的信息量不同。
```

请写：

```text
抽检10件中0件不合格，能说明：
抽检100件中0件不合格，能说明：
```

参考表达：

```text
抽检10件中0件不合格，只能说明这10件没有发现不合格。
抽检100件中0件不合格，提供更强证据，但仍依赖随机抽样和独立性。
```

此刻你应该能说清：

```text
参数控制分布形状；样本量控制我们能看到多少信息。
```

## 换成不同过程

不同工业问题对应不同分布模型。

| 问题 | 随机变量 | 常用模型 | 直觉 |
| --- | --- | --- | --- |
| 10件中不合格件数 | 次数 | 二项分布 | 固定次数里数成功或失败 |
| 一平方米表面缺陷数 | 计数 | 泊松分布 | 固定区域里数稀疏事件 |
| 零件直径测量误差 | 连续值 | 正态分布 | 多个小误差叠加 |
| 设备无故障运行时间 | 时间 | 指数或威布尔分布 | 等待失效发生 |

请任选一行，写成完整句子：

```text
我研究的是：
随机变量是：
分布模型表达的是：
```

参考表达：

```text
我研究的是表面缺陷。
随机变量是每平方米表面缺陷数。
分布模型表达的是单位面积内缺陷出现次数的生成机制。
```

此刻你应该能说清：

```text
选择分布前，先说清随机变量和生成过程。
```

## 比较模型

现在看四种模型的直觉，不推导公式。

### 伯努利分布

```text
一次试验，只有两种结果。
例子：单件产品是否合格。
```

板书：

```text
X = 1 表示合格
X = 0 表示不合格
P(X=1)=p
```

### 二项分布

```text
固定 n 次独立试验，数某个结果出现几次。
例子：10件里不合格件数。
```

板书：

```text
X ~ Binomial(n,p)
E(X)=np
```

### 泊松分布

```text
固定空间或时间内，数稀疏事件出现几次。
例子：每平方米表面缺陷数。
```

板书：

```text
X ~ Poisson(λ)
E(X)=λ
```

### 正态分布

```text
连续测量值围绕中心上下波动。
例子：零件直径、材料强度、测量误差。
```

板书：

```text
X ~ Normal(μ, σ)
μ 表示中心位置
σ 表示波动大小
```

此刻你应该能说清：

```text
分布名称背后是不同的数据生成机制。
```

## 迁移到工业问题

某材料关键性能指标目标值为 100。历史数据显示：

```text
均值 μ = 100
标准差 σ = 2
```

如果用正态模型描述：

```text
X ~ Normal(100, 2)
```

请回答：

```text
X 表示什么：
100 表示什么：
2 表示什么：
这个模型不能说明什么：
```

参考表达：

```text
X 表示某件材料的关键性能指标。
100 表示性能指标中心水平。
2 表示性能指标的典型波动大小。
这个模型不能自动说明波动来源，也不能证明过程一直稳定。
```

现在加入规格：

```text
合格范围：96 到 104
```

请写判断：

```text
如果 σ 从2变成4，均值仍为100，风险怎样变化：
```

参考表达：

```text
均值仍在中心，但波动变大，落到规格外的风险上升。
```

此刻你应该能说清：

```text
分布让我们同时表达中心、波动和越界风险。
```

## 划清边界

分布模型能帮助我们：

```text
描述取值范围、常见取值、尾部风险和长期平均。
```

分布模型不能自动保证：

```text
数据独立、过程稳定、模型形式正确、参数估计准确。
```

请写下边界句：

```text
拟合出一个分布，不等于现实过程已经被正确理解。
```

现在完成最终输出：

```text
随机变量：
分布模型：
模型解释：
边界说明：
```

参考表达：

```text
随机变量：X = 10件产品中的不合格件数。
分布模型：X ~ Binomial(10, 0.05)。
模型解释：在独立且不合格率为5%的假设下，描述0到10件不合格的概率。
边界说明：结论依赖独立性、抽样方式和不合格率稳定。
```

此刻你应该能说清：

```text
分布是有条件的模型语言，不是现实的保证书。
```

## 连接下一课

今天我们写了模型：

```text
X ~ Binomial(n,p)
X ~ Poisson(λ)
X ~ Normal(μ,σ)
```

但参数从哪里来？

```text
p、λ、μ、σ 可以来自历史数据、专家经验、物理机理或新样本。
```

下一课要问：

```text
如果先有一个参数判断，又看到新数据，怎样连续更新？
```

这会把第 4 次贝叶斯和本节分布连接起来。

此刻你应该能说清：

```text
第4次更新事件可信程度；下一课开始更新模型参数和分布判断。
```

## 收束：用三句话带走今天

第一句：一句模型结论。

```text
X = 10件中的不合格件数，可以用二项分布描述。
```

第二句：一句边界说明。

```text
这个模型依赖固定不合格率、独立抽样和过程稳定。
```

第三句：一个下一课问题。

```text
当新抽检数据出现以后，怎样更新我们对不合格率p的判断？
```

## 本节板书总表

| 名称 | 直观含义 | 本节表达 |
| --- | --- | --- |
| 随机变量 | 把随机结果变成数字 | `X=10件中不合格件数` |
| 取值范围 | 变量能取哪些值 | `0,1,...,10` |
| 分布 | 每个取值及其概率 | `P(X=k)` |
| 参数 | 控制分布形状的量 | `n,p,λ,μ,σ` |
| 期望 | 长期平均结果 | `E(X)=np` |
| 二项分布 | 固定次数中数事件 | `Binomial(n,p)` |
| 泊松分布 | 固定区域中数稀疏事件 | `Poisson(λ)` |
| 正态分布 | 连续值围绕中心波动 | `Normal(μ,σ)` |
| 模型边界 | 分布成立所依赖的条件 | 独立性、稳定性、抽样方式 |
