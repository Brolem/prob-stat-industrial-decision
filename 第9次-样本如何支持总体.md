## 开场：样本不是总体

前面课程中，我们不断使用数据更新判断。

今天只问一个基础问题：

```text
样本怎样支持总体结论？
```

请写下今天的课堂契约：

```text
样本给证据，总体是目标；二者之间需要推断桥梁。
```

今天的核心句是：

```text
点估计告诉我看到什么，区间和检验告诉我不确定性有多大。
```

此刻你应该能说清：

```text
样本比例不是总体比例本身。
```

---

## 先做判断

某供应商一批产品共 10000 件。入厂抽检 50 件，发现 2 件不合格。

管理者问：

```text
这批产品的不合格率是否可以认为低于5%？
```

请先写：

```text
第一次判断：低于5% / 不能判断 / 不支持低于5%
理由：
我最关注的数字：
```

停 1 分钟。

现在回答：

```text
样本不合格率 = 2 / 50 = 4%
```

请不要立刻说总体就是 4%。

写下：

```text
样本不合格率是总体不合格率的估计，不是总体不合格率本身。
```

此刻你应该能说清：

```text
4%是点估计，还没有表达抽样波动。
```

---

## 看抽样波动

假设真实总体不合格率就是 5%。不同抽检 50 件，会得到不同不合格件数。

下面是一组模型下的常见结果：

| 样本不合格件数 | 样本不合格率 | 在真实5%下是否常见 |
| ---: | ---: | --- |
| 0 | 0% | 会出现 |
| 1 | 2% | 会出现 |
| 2 | 4% | 会出现 |
| 3 | 6% | 会出现 |
| 4 | 8% | 会出现 |
| 5及以上 | 10%及以上 | 较少见但仍会出现 |

请回答：

```text
如果真实不合格率是5%，抽到2/50是否与5%明显冲突：
```

参考回答：

```text
不明显冲突。真实5%时，50件里抽到2件不合格是正常抽样波动内的结果。
```

现在改写判断：

```text
第二次判断：
我改变判断的原因：
```

此刻你应该能说清：

```text
样本低于5%，不等于总体已经低于5%。
```

## 写区间

区间估计的作用是：

```text
用一段范围表达总体参数的不确定性。
```

对 50 件中 2 件不合格，可以给一个粗略表达：

```text
样本点估计为4%，但总体不合格率仍有明显不确定性。
```

这里不用复杂公式，先学表达结构：

```text
点估计：
不确定性：
结论边界：
```

请写：

```text
点估计：样本不合格率为4%。
不确定性：样本量只有50件，抽样波动较大。
结论边界：不能仅凭2/50断定总体不合格率低于5%。
```

现在比较两个样本：

| 抽检结果 | 样本不合格率 | 结论强度 |
| --- | ---: | --- |
| 2/50 | 4% | 证据较弱 |
| 20/500 | 4% | 证据更强 |

请回答：

```text
为什么同样4%，500件比50件证据更强：
```

参考表达：

```text
样本量更大，抽样波动更小，点估计更稳定。
```

此刻你应该能说清：

```text
同一个样本比例，样本量不同，证据强度不同。
```

## 理解检验

假设工厂有一个基准说法：

```text
总体不合格率为5%。
```

现在样本是：

```text
50件中2件不合格。
```

检验思路不是问“样本是不是4%”。样本已经是 4%。

检验问的是：

```text
如果总体真是5%，抽到2/50这样的结果是否反常？
```

如果不反常：

```text
样本没有提供足够理由推翻5%基准。
```

如果很反常：

```text
样本支持总体已经偏离5%基准。
```

请写下：

```text
p值的直观问题：
如果基准模型成立，看到当前或更极端数据的机会有多大？
```

此刻你应该能说清：

```text
检验不是证明基准为真，而是检查数据是否与基准冲突。
```

## 比较两种语言

同一份数据可以用两种统计语言表达。

数据：

```text
抽检50件，2件不合格。
```

频率学派语言：

```text
如果重复使用这个抽样和区间程序，长期有固定比例的区间覆盖真实总体参数。
```

贝叶斯语言：

```text
在先验和当前样本下，总体不合格率落在某个范围内的可信程度是多少。
```

请把差异写成表：

| 问题 | 频率学派语言 | 贝叶斯语言 |
| --- | --- | --- |
| 参数是什么 | 固定未知量 | 不确定判断对象 |
| 概率说什么 | 程序长期表现 | 当前可信程度 |
| 区间怎样解释 | 程序生成的区间 | 参数可信范围 |

此刻你应该能说清：

```text
两种语言回答的问题不同，不能把解释混在一起。
```

## 迁移到工业抽检

现在给出一份工业数据：

```text
新供应商首批抽检200件，发现6件不合格。
样本不合格率为3%。
历史同类供应商不合格率为2%。
放行阈值为5%。
```

请写四句话：

```text
样本结论：
与历史比较：
总体边界：
行动建议：
```

参考表达：

```text
样本结论：本次抽检200件中6件不合格，样本不合格率为3%。
与历史比较：高于历史同类供应商2%的水平。
总体边界：样本仍有抽样波动，不能把3%直接当成总体不合格率。
行动建议：暂不直接拒收，但建议增加抽检并追踪后续批次。
```

此刻你应该能说清：

```text
工业抽检结论要同时包含样本结果、总体不确定性和行动建议。
```

## 划清边界

样本推断的边界：

| 边界 | 说明 |
| --- | --- |
| 抽样方式 | 样本是否代表总体 |
| 样本量 | 抽样波动大小 |
| 测量误差 | 检测本身是否可靠 |
| 批次异质性 | 总体内部是否分层 |
| 时间变化 | 当前批次是否代表未来批次 |

请写下边界句：

```text
样本支持总体，需要代表性、样本量和测量可靠性共同支撑。
```

完成最终输出：

```text
样本结论：
总体推断：
边界说明：
下一步数据：
```

此刻你应该能说清：

```text
样本不是总体，但好的抽样设计能让样本成为总体判断的证据。
```

## 连接下一课

今天我们讨论：

```text
样本怎样支持总体。
```

下一课要问：

```text
即使样本和总体关系说清了，证据能不能说明因果？
```

例如：

```text
换了工艺后，不合格率下降。
这是否证明新工艺导致质量改善？
```

这会进入实验设计和因果识别。

此刻你应该能说清：

```text
统计证据可以支持差异，但因果结论还需要设计支撑。
```

## 收束：用三句话带走今天

第一句：一句样本结论。

```text
50件抽检中2件不合格，样本不合格率为4%。
```

第二句：一句边界说明。

```text
4%是总体不合格率的点估计，不是总体本身，样本量会影响证据强度。
```

第三句：一个下一课问题。

```text
看到工艺改变后质量改善，怎样判断这是不是因果作用？
```

## 本节板书总表

| 名称 | 直观含义 | 本节表达 |
| --- | --- | --- |
| 样本 | 实际观察到的数据 | 50件中2件不合格 |
| 总体 | 要判断的对象 | 10000件产品 |
| 点估计 | 用一个数估总体 | `2/50=4%` |
| 抽样波动 | 同一总体会抽出不同样本 | 5%总体下可抽到2/50 |
| 区间 | 用范围表达不确定性 | 总体不合格率范围 |
| 检验 | 数据是否与基准冲突 | 基准为5%时2/50是否反常 |
| 频率语言 | 程序长期表现 | 重复抽样覆盖率 |
| 贝叶斯语言 | 当前可信程度 | 参数落入范围的可信度 |
